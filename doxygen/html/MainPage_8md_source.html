1 # OscProb {#index}
2 
3 OscProb is a small set of classes aimed at computing exact neutrino oscillation probabilities with a few different models.
4 
5 OscProb contains a basic framework for computing neutrino oscillation probabilities. It is integrated into [ROOT](https://root.cern.ch/), so that each class can be used as you would any ROOT class.
6 
7 Available classes are:
8 - **[PremModel](@ref OscProb::PremModel):** Used for determining neutrino paths through the earth
9 - **[PMNS_Fast](@ref OscProb::PMNS_Fast):** Standard 3-flavour oscillations
10 - **[PMNS_Sterile](@ref OscProb::PMNS_Sterile):** Oscillations with any number of neutrinos
11 - **[PMNS_NSI](@ref OscProb::PMNS_NSI):** Oscillations with 3 flavours including Non-Standard Interactions
12 - **[PMNS_Deco](@ref OscProb::PMNS_Deco):** Oscillations with 3 flavours including a simple decoherence model
13 
14 A few example macros on how to use OscProb are available in a tutorial directory.
15 
16 # Installing OscProb
17 
18 OscProb is very easy to install. The only requirements is to have ROOT installed with the GSL libraries.
19 
20 **NEW: Thanks to Jacek Holeczek, OscProb now also builds with ROOT 6!!**
21 
22 Once you have ROOT setup, simply do:
23 ```sh
24 cd OscProb
25 make
26 ```
27 
28 A shared library will be produced: ```libOscProb.so```
29 
30 This should take a few seconds and you are all set.
31 
32 Just load the shared library in your ROOT macros with:
33 ```cpp
34 gSystem->Load("/full/path/to/libOscProb.so");
35 ```
36 
37 In some cases you may need to explicitly load GSL libraries. Just add something like this to your rootlogon, replacing the path to the GSL libraries if needed:
38 ```cpp
39 gSystem->Load("/usr/lib/x86_64-linux-gnu/libgsl.so");
40 gSystem->Load("/usr/lib/x86_64-linux-gnu/libgslcblas.so");
41 ```

43 # Tutorial
44 
45 In the directory OscProb/tutorial you will find a few macros with examples using OscProb.
46 
47 Two macros are particularly useful:
48 - ```simpleExamples.C``` : Contains some short pieces of code on how to perform different tasks.
49 - ```MakeOscillogram.C``` : Runs a full example of how to plot an oscillogram with the PREM model.
50 
51 Additionally, the macro ```SetNiceStyle.C``` will provide simple tools to make your plots look nicer. Feel free to use it anytime you're making plots, even if you're not running OscProb. This is completely independent of OscProb.
