Detailed Description

This class expands the PMNS_Fast class including a general NU mixing matrix

The non unitarity effect is parametrized by dimensionless quantities alpha which quantify the deviation from unitarity with respect to the standard mixing

Reference: https://arxiv.org/pdf/2111.00329.pdf
